# 원터치 복사 발송 (Copy Message) — 사장님 사용법

사장님이 봇에게 보낸 **영상/이미지+캡션** 메시지를 그대로 2만 명에게 `copy_message`로 복사 발송하는 시스템입니다.

---

## 1️⃣ 장전하기 (발송할 메시지 저장)

1. **봇과 1:1 채팅**을 엽니다.
2. **영상** 또는 **이미지**를 보냅니다.  
   - 캡션(텍스트)을 붙여도 됩니다.  
   - 동영상 파일로 보내도 인식됩니다.
3. 봇이 **"✅ 장전 완료 (영상)"** 또는 **"✅ 장전 완료 (이미지)"** 로 답하면, 그 메시지가 **장전된 메시지**로 저장된 상태입니다.  
   - 마지막으로 장전한 메시지 하나만 유지됩니다. (다시 장전하면 덮어쓰기)

---

## 2️⃣ 테스트 (선택)

1. 봇에게 **`/admin`** 을 입력합니다.
2. **"🧪 장전된 메시지 테스트 (나에게만)"** 버튼을 누르면, 장전된 메시지가 **사장님에게만** 1건 복사 발송됩니다. (VIP CASINO 버튼 동일 적용)
3. 최종 결과물을 확인한 뒤 발사하세요.

## 3️⃣ 발사하기 (전체 유저에게 발송)

1. 봇에게 **`/admin`** 을 입력합니다.
2. **"🚀 장전된 메시지 발사"** 버튼을 누릅니다.
4. 봇이 **"📤 발사 시작..."** 이라고 하고,  
   **500명 단위**로 발송하면서 **진행 상황**을 알려줍니다.  
   - 예: `📤 청크 발송 완료 500/20000 (성공 500, 실패 0)`
5. 끝나면 **"✅ 발사 완료. 총 N명 중 성공 M, 실패 K"** 로 요약이 옵니다.

---

## 자동 적용 사항

- **VIP CASINO 버튼**: 복사된 메시지 아래에 **VIP CASINO** 인라인 버튼이 자동으로 붙습니다. (연결 주소: `GAME_PAGE_URL`)
- **안전 발송**: 500명 단위 청크, 청크마다 1.2초 대기. 일부 유저 발송 실패 시에도 나머지는 계속 발송됩니다.

---

## 필요한 설정

- **ADMIN_ID**: 사장님 텔레그램 user_id (장전/발사는 이 계정만 가능).
- **GAME_PAGE_URL**: VIP CASINO 버튼 링크.
- **users 테이블**: `data/users.db` 에 발송 대상 유저 `user_id` 가 있어야 합니다. (봇과 대화한 유저가 자동 저장됨)
